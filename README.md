# Система учёта домашних животных

## Описание проекта

Данная программа представляет собой **интерактивную систему для управления информацией о владельцах, их питомцах, зоомагазине и ветеринарной клинике**. <br>
Проект выполнен в **объектно-ориентированном стиле** на языке **Python** с использованием **датаклассов**, **аннотаций типов**, **обработки исключений** и **работы с файлами JSON и XML** для хранения данных.

Программа имитирует работу целой экосистемы, где:

* Владельцы могут иметь питомцев, просматривать их паспорта и медкарты.
* Зоомагазин продаёт питомцев владельцам.
* Ветеринарная клиника лечит животных и ведёт историю обращений.

## Цель работы

Разработка программного обеспечения, реализующего учёт домашних животных, включая хранение, просмотр и редактирование данных о владельцах, питомцах, ветеринарных записях и продажах животных.

## Основной функционал

### Владельцы

* Добавление, просмотр и удаление владельцев.
* Управление питомцами владельца (добавление, удаление, просмотр паспортов и медкарт).
* Отправка питомца в ветклинику для лечения или вакцинации.

### Зоомагазин

* Добавление новых питомцев в магазин.
* Просмотр доступных животных.
* Продажа питомца владельцу.
* Ведение журнала продаж.

### Ветеринарная клиника

* Добавление записей о болезнях и прививках.
* Просмотр всех медицинских карт питомцев.
* Хранение истории посещений клиники.

### Сохранение данных

* Автоматическая сериализация всех данных в **JSON** и **XML** файлы.
* Загрузка сохранённых данных при запуске программы.
